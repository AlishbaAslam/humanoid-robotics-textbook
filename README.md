# Humanoid Robotics Textbook

This repository contains educational materials for learning about humanoid robotics, covering ROS2 middleware, digital twin simulation, AI robot brains, and voice-language-action (VLA) educational modules.

## Project Structure

- **`.specify/`** - SpecKit Plus templates and scripts for project management
- **`specs/`** - Feature specifications, plans, and tasks
- **`website/`** - Docusaurus-based documentation website
- **`examples/`** - Code examples for various robotics concepts
- **`history/`** - Prompt History Records for project history

## Modules

### Module 1: ROS2 Middleware
- Basic ROS2 concepts and publisher/subscriber patterns
- Python agents and sensor integration
- URDF modeling and robot description

### Module 2: Digital Twin Simulation
- Gazebo simulation environment
- Unity visualization
- Sensor simulation (LIDAR, IMU, depth cameras)

### Module 3: AI Robot Brain
- Isaac Sim simulation environment
- Isaac ROS integration
- Navigation 2 (Nav2) for humanoid robots

### Module 4: Voice-Language-Action (VLA) Educational Module
- Voice-to-action processing
- Cognitive planning
- Capstone project integration

## Getting Started

1. Navigate to the website directory:
   ```bash
   cd website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Contributing

This project uses Spec-Driven Development (SDD) methodology with:
- Feature specifications in `specs/`
- Implementation plans in `specs/plan.md`
- Task breakdowns in `specs/tasks.md`
- Prompt History Records in `history/prompts/`
